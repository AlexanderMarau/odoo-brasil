## Funcionalidades

* Cálculo de impostos (ICMS, IPI, PIS, COFINS, ISSQN, DIFAL, FCP, Simples Nacional)
* Emissão de NFe
* Emissão de Boletos
* Integração Bancária (cnab240)
* Folha de Pagamento
* Importação de extratos bancários
* Contas a pagar e Receber
